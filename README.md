# SmartHostelManagementSystem (MVC)

This is a clean ASP.NET Core 10.0 MVC project scaffolded with the default template.

## Structure
- Controllers/
- Models/
- Views/
- wwwroot/
- Program.cs
- Startup.cs
- appsettings.json

## Getting Started

1. Restore dependencies:
   ```sh
   dotnet restore
   ```
2. Build the project:
   ```sh
   dotnet build
   ```
3. Run the project:
   ```sh
   dotnet run
   ```
4. Open your browser to `http://localhost:5000` (or the port shown in the console).

---

This project contains no custom code or features. Use it as a starting point for your ASP.NET Core MVC development.
